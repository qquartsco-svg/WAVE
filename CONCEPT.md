# Wave Propagation Foundation — Concept

파동 방정식 하나에서 모든 것이 나온다:

```
∂²u/∂t² = c²∇²u
```

- c를 바꾸면 음파, 지진파, 수면파, 빛, 중력파가 된다.
- 감쇠를 넣으면 진동 붕괴가 된다.
- 비선형 항을 넣으면 충격파가 된다.

이 foundation은 그 출발점:
1. 일반 파동 → 파장, 주파수, 에너지, 감쇠
2. 진동자 → 공명, Q-factor, 붕괴
3. 지진파 → P/S/표면파 추적, 진앙 삼각측량
4. 수면파·음파 → 분산관계, 심해/천해, 음향 임피던스
5. 중력파 → strain, chirp mass, LIGO/LISA 감도

현재 단계는 **screening foundation** — 정밀 수치 시뮬은 아니지만, 파동 현상의 가능성과 특성을 빠르게 읽는다.
